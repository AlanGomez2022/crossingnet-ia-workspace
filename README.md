# CrossingNet HTML Workspace

Workspace para generar bloques HTML de newsletters de CrossingNet usando Codex en VS Code.

Este proyecto NO procesa imágenes.

Las imágenes se preparan manualmente en Photoshop:
- ancho: 760px;
- división en bloques;
- peso máximo sugerido por imagen: 50KB.

Codex solo debe generar el código HTML final respetando las reglas del proyecto.
